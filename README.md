# PyRPGame
A simple, text based RPG built in Python. This is a no copy without permission repo, so please, do not download or dump this repo online!

Instructions:

Place the PyRPG folder in your AppData directory (Press the Windows key, type %appdata%, and press Enter.)

Paste the PyRPG folder there.

Inside the PyRPG folder, create a folder named saves.

Make sure you have Python 3.13.7 or newer installed before running the game.

You need a code editor (like VS Code or Cursor) to run this program properly.
